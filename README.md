# EchoRead 

EchoRead is an adaptive AI voice narrator for light novels.
It analyzes story structure and emotion, then generates expressive narration
that adapts in real time.

## Tech Stack
- FastAPI (backend)
- Gemini (story analysis)
- ElevenLabs (voice synthesis)
- Next.js + Tailwind (frontend)

## Features
- Emotion-aware narration
- Adaptive pacing and tone
- Role-based narration (narrator / dialogue)
- Clean web interface

## Demo
Paste a short light novel passage and click **Narrate**.
